What is this?

This is part of an ongoing experiment to get an idea of what local employers are looking for as far as tech goes. All of the data was pulled from Indeed.com, so there is certainly potential for skewed results. However, Indeed's popularity should mean the results are fairly representative of Nashville's tech scene.

How was this done?

Google Sheets! I only found out recently that G Sheets can not only be scripted with JavaScript (as well as the rest of G Suite), it can also be used as a web scraper. So I set up my scraper to run on a weekly basis and dump the results into a spreadsheet. That sheet is then accessed by this site (as JSON) and then put through a pipeline of functions to bring you the tidy results you see here.

Careful now...

"*" means questionable results. All terms with filtered out with regular expressions. Unfortunately, regexs don't allow for context, so non-tech specific words like "go" or letters like "c" can yield a lot of false positives.
